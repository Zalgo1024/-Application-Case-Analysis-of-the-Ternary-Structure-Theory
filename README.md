# 三元结构理论 案例分析引擎

基于**三元结构理论（Triadic Structure Theory）**的自动化案例分析系统。以"生存—繁衍—利益"三维框架为分析工具，输入社会/司法/历史事件，输出结构化六部分分析报告（封面页 → 分析框架前置说明 → 概念解读 → 案例事实摘要 → 三元结构分析正文 → 结论）。

## 核心功能

- **多维分析引擎**：自动识别事件中涉及的生存、繁衍、利益维度冲突
- **利益网络可视化**：生成交互式利益动线图与叙事战场时间线
- **标准化报告输出**：输出 Word (.docx) + PDF 双格式，遵循体制内公文排版规范
- **配图溯源机制**：支持编码生图（DIAGRAM → SVG/PNG）与网络素材链式降级采集（Wikimedia → 官媒 → Unsplash）

## 项目结构

```
├── analysis_engine.py          # 分析引擎核心（解析 → 分析 → 导出）

├── image_sourcing.py           # 图片素材溯源管理器
├── theory_config.json          # 理论维度配置
├── analysis_prompt.md          # 分析提示词模板
├── 三元结构理论 案例分析本体.md   # 三元结构理论本体说明
├── 三元结构理论 利益分析 Skill.md # 利益分析 Skill 说明
├── run_report_*.py             # 各案例报告生成脚本
├── 报告/                       # 生成的报告输出目录

```
